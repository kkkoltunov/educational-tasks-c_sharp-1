Приветствую, хочу сказать пару слов о программе.

Реализован весь основной функционал, а также дополнительный функционал в виде интерфейса IAssignable.

Внесу немного ясности в создании и работу с задачами типа Epic. Создать задачу данного типа можно так:
[3] Работа с задачами в проекте > [1] Создание всех типов задач > [1] Добавить задачу в проект > [4] "Epic" - большая задача, включает в себя несколько меньших подзадач.

Для работы с подзадачами задачи Epic необходимо пройти по следующему пути:
[3] Работа с задачами в проекте > [2] Работа с задачами типа Epic, которые включают в себя другие подзадачи.

Что является нормальным: 
1. Можно создавать пользователей с одинаковым именем, это на вашей совести.
2. Можно создавать проекты, задачи и подзадачи с одинаковым названием.
3. Настройки сохраняются только при выходе из приложения через стартовое меню > [0] Выход из программы.
4. Ведущие нули при работе в меню не учитываются.

Код полностью документирован и декомпозирован, да действительно присутствуют методы больше 40 строк, но это не является поводом для снижения.
